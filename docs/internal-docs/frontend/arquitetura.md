# Arquitetura

## Feature-Sliced Design + Server Actions

Este projeto utiliza:

- Next.js (App Router)
- ShadCN/UI
- TypeScript
- Server Actions

### Estrutura de Pastas

```
src/
├── app/              # Rotas da aplicação
├── components/       # Componentes compartilhados
├── features/         # Módulos de funcionalidades e seus componentes
├── lib/              # Utilitários e configurações
├── styles/           # Estilos globais (talvez não seja usado)
└── types/            # Types globais
```
